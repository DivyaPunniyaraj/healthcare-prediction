## Objective
Perform exploratory data analysis on the PIMA Indians Diabetes Dataset before training the machine learning model.

## Dataset
- diabetes.csv
- Source: PIMA Indians Diabetes Dataset

## Steps Performed
- Imported required libraries
- Loaded dataset
- Checked dataset shape
- Displayed dataset information
- Checked missing values
- Generated summary statistics
- Visualized distributions
- Created BMI Category feature
- Encoded BMI Category
- Selected features
- Scaled features
- Split dataset into training and testing sets
 ### Dashboard visualization

This dashboard provides a visual overview of the diabetes dataset used to train the Machine Learning model. It presents important healthcare statistics and graphical insights that help users understand patient characteristics and the distribution of diabetes cases.

The dashboard enables quick analysis of patient information through interactive charts and summary metrics, supporting data-driven healthcare decisions
